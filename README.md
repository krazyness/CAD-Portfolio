CAD Portfolio — Static homepage

This repository contains a minimal static homepage for a CAD portfolio suitable for hosting on GitHub Pages.

What's included

- index.html — the homepage with sections: hero, projects, about, contact, and footer.
- css/styles.css — responsive styles and variables for easy theming.

How to preview locally

Open `index.html` in a browser. For a simple local server (recommended) you can run from the repository root:

# Python 3.x
python -m http.server 8000

Then open http://localhost:8000 in your browser.

Deploying to GitHub Pages

1. Create a repository on GitHub named `your-username.github.io` (or any repo and enable Pages from the main branch).
2. Commit and push these files to the repository's `main` branch.
3. Enable GitHub Pages in the repository settings and choose `main`/`root` if needed.

Customization

- Replace placeholder text (like `[Your Name]`) with your details.
- Replace SVG placeholders or card links with images and case study pages.
- Update contact links and add a projects folder for full case studies.
