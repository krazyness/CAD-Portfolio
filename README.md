CAD Portfolio — Static homepage

This repository contains a minimal static homepage for a CAD portfolio suitable for hosting on GitHub Pages.

What's included

- index.html — the homepage with sections: hero, projects, about, contact, and footer.
- css/styles.css — responsive styles and variables for easy theming.

How to preview locally

Open `index.html` in a browser. For a simple local server (recommended) you can run from the repository root:

# Python 3.x
python -m http.server 8000

Then open http://localhost:8000 in your browser.
